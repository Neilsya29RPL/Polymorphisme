Pada hasil run latihan 2 terdapat panjang garis A dan garis B yang  diperoleh dengan cara memanggil method getLength() pada class Line. Pada hasil run tersebut menunjukkan bahwa garis A lebih kecil dari pada garis B (A < B) atau garis B lebih lebih besar dari pada garis A (B > A). Maka pernyataan-pernyataan tersebut bernilai TRUE jika memenuhi atau sesuai dengan panjang garis yang diperoleh dan bernilai FALSE jika pernyataan tersebut tidak memenuhi atau tidak sesuai.
